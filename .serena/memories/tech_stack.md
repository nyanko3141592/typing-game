# Technical Stack

## Frontend Technologies
- **HTML5**: Semantic markup with Japanese language support
- **CSS3**: Modern CSS with Grid, Flexbox, and animations
- **Vanilla JavaScript**: No frameworks or build tools - pure ES6+ JavaScript

## Styling Approach
- **CSS Grid & Flexbox**: For responsive layouts
- **CSS Animations**: Smooth transitions and visual feedback
- **Custom CSS Variables**: For consistent theming
- **Mobile-first responsive design**: Works on both desktop and mobile

## Data Management
- **LocalStorage**: For rankings and user data persistence
- **JSON**: Static data files (questions.json)
- **No backend**: Fully client-side application

## Deployment & Hosting
- **GitHub Pages**: Static site hosting
- **No build process**: Direct deployment of source files
- **No package manager**: No npm/yarn dependencies

## Browser Compatibility
- Modern browsers with ES6+ support
- IME compatibility for Japanese input
- Mobile browser support for touch interactions

## Architecture
- **Single Page Application**: Multiple screens managed via JavaScript
- **Event-driven**: DOM event listeners for user interactions
- **Modular JavaScript**: Functions organized by feature area
- **Separation of concerns**: HTML structure, CSS styling, JS behavior
